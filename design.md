# Design System: Pyro-Data Insight (Hanwha Aerospace Inspired)

## Visual Identity
Inspired by the Hanwha Aerospace brand, focusing on precision, technology, and reliability.

## Color Palette
- **Primary (Hanwha Orange)**: `#F37321` - Used for key actions, status indicators, and branding accents.
- **Secondary (Aerospace Charcoal)**: `#1C1C1C` - Used for sidebars, headers, and primary text for a professional feel.
- **Accent (Technical Blue)**: `#0067B2` - Used sparingly for links and secondary highlights.
- **Background (Neutral Gray)**: `#F8F9FA` - Clean workspace background.
- **Success (Precision Green)**: `#28A745`
- **Warning (Alert Amber)**: `#FFC107`

## Typography
- **Display**: 'Space Grotesk' - For headers and model names, giving a modern engineering look.
- **Body**: 'Inter' - For general UI and descriptions.
- **Data**: 'JetBrains Mono' - For all numerical values and pressure readings to emphasize technical accuracy.

## UI Components
- **Borders**: 1px solid `#E0E0E0` for light mode, `#333333` for dark mode.
- **Shadows**: Minimal, focused on "layering" rather than depth.
- **Corner Radius**: 8px (Base), 16px (Cards) - Balancing approachability with industrial structure.
